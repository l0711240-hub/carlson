# 🎓 인하대학교 사회교육과 먹이사슬

학부생으로 시작해 **사회 교사 또는 교수**가 되는 학업 시뮬레이션 + 먹이사슬 액션 게임.
원래 단일 HTML(약 11,000줄)이던 코드를 **수정·열람이 쉽도록 여러 파일로 분리**한 버전입니다.

---

## ▶️ 실행 방법

### GitHub Pages (권장)
1. 이 폴더 전체를 GitHub 저장소에 올립니다 (`index.html`이 최상위에 있어야 함).
2. 저장소 **Settings → Pages → Branch**를 `main / (root)`으로 설정.
3. 잠시 후 `https://<아이디>.github.io/<저장소명>/` 에서 실행됩니다.

### 로컬 실행
- `index.html`을 더블클릭하면 브라우저에서 바로 실행됩니다. (별도 서버 불필요)

---

## 📁 폴더 구조

```
inha-social-edu/
├── index.html              ← 게임 화면(HTML) + 스크립트 로드 순서
├── css/
│   └── styles.css          ← 모든 디자인/스타일
├── js/
│   ├── data/               ← ★ 과목·스킬 데이터 (가장 자주 수정)
│   ├── systems/            ← 수강신청·졸업·대학원·진로 등 게임 로직
│   └── engine/             ← 게임 루프·보스 전투·입력·렌더링
└── docs/                   ← 코드 수정 가이드 · 게임 플레이 가이드
```

> ⚠️ `index.html`의 `<script>` **로드 순서를 바꾸지 마세요.** 파일들은 순서대로 실행되며 앞 파일이 뒤 파일의 토대가 됩니다. (새 파일을 추가할 땐 알맞은 위치에 `<script src>` 한 줄을 넣으세요.)

## ✍️ 수정 규칙 (꼭 지키기)

1. 수정 전 **백업**(저장소를 쓰면 커밋 단위로 자동 보존).
2. 과목의 **`code`(학수번호)는 고유**해야 하며, 함부로 바꾸면 스킬·보스 연결이 끊깁니다. 이름만 바꾸려면 `name`만 수정.
3. 파일은 **UTF-8**로 저장(한글 깨짐 방지).
4. 저장 후 브라우저 **새로고침(Ctrl+F5)**. 안 켜지면 **F12 → Console**의 빨간 오류 확인.
5. 자세한 과목·카테고리 편집법은 **`docs/코드_수정_가이드.md`** 참고.

---

## 🧩 파일 분리 방식 (개발자용)

- 모든 JS는 **클래식 스크립트**(`<script src>`)로, 전역 스코프와 실행 순서를 공유합니다. 원본 단일 스크립트와 **동작이 100% 동일**합니다.
- 그래서 **`type="module"`로 바꾸거나 순서를 섞으면 안 됩니다.**
- 단일 파일로 다시 합치고 싶으면, `js/` 안의 파일을 `index.html`의 순서대로 이어 붙이면 됩니다. (원본은 별도 보관)

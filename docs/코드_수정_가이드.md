# 🛠️ 인하대 사회교육과 먹이사슬 — 코드 수정 가이드

이 문서는 `inha_social_edu.html` 파일을 직접 고쳐서 과목·교수·졸업요건 등을 바꾸고 싶을 때 보는 안내서입니다.

---

## 0. 시작하기 전에

| 항목 | 내용 |
|---|---|
| 편집 도구 | **VS Code**(권장), 메모장++ 등 텍스트 에디터 |
| 실행 | HTML 파일을 더블클릭 → 브라우저에서 바로 실행 |
| 수정 반영 | 저장 후 브라우저 **새로고침(Ctrl+F5)** |
| 오류 확인 | 게임이 안 켜지면 브라우저 **F12 → Console** 탭의 빨간 메시지 확인 |
| **백업** | 수정 전 **반드시 파일을 복사**해 두세요 (예: `inha_social_edu_backup.html`) |

> 이 게임은 **단일 HTML 파일** 하나로 되어 있고, 모든 로직은 파일 안의 단 하나의 `<script>` 블록에 들어 있습니다. 별도 설치·서버가 필요 없습니다.

---

## 1. 가장 중요한 개념 — "코드(학수번호) = 모든 것의 키"

이 게임의 모든 시스템은 과목의 **`code`(학수번호)** 를 열쇠로 서로 연결됩니다.

```
과목 정의 ──(code)──┬── 스킬 부여 (COURSE_SKILL_MAP)
                    ├── 교수 보스 등장 (BOSS_DEFS)
                    ├── 복수전공 인정 (크로스오버 코드 목록)
                    └── 졸업요건 체크 (checkGraduationReady)
```

➡️ **규칙 1: `code`는 절대 중복되면 안 됩니다.**
➡️ **규칙 2: `code`를 바꾸면 그 코드를 참조하던 연결이 모두 끊어집니다.** 이름만 바꾸고 싶으면 `name`만 고치세요.

## 2. 과목 시스템

### 2-1. 과목 배열은 어디에 있나
파일에서 아래 이름을 **검색(Ctrl+F)** 하면 바로 찾습니다. (줄 번호는 수정에 따라 달라지므로 검색을 권장합니다.)

| 검색어 | 배열 | 역할 |
|---|---|---|
| `const LIBERAL_COURSES` | 교양 | 기초·중점/핵심/일반교양 |
| `const GEO_COURSES` | **주전공(사회교육과)** | 전공필수·전공선택 |
| `const AGRI_COURSES` | **교직과정** | 교직이론·소양·실습 |
| `const ECON_COURSES` 등 | 복수전공/부전공·타 학과 | 경제·정치·사회 등 |
| `ALL_COURSES_WITH_POLISCI` | 통합 배열 | 위 배열을 전부 합치는 곳 |

### 2-2. 과목 한 줄의 구조
```js
{code:"208.205", name:"경제와 사회", credit:3, category:"geo", type:"normal", semester:[1], recPick:3},
```

| 필드 | 의미 | 비고 |
|---|---|---|
| `code` | 학수번호(고유 키) | **중복 금지** |
| `name` | 화면에 보이는 과목명 | 가장 안전하게 바꿀 수 있음 |
| `credit` | 학점 | 숫자 |
| `category` | 소속 트랙(=어느 탭에 뜰지) | `liberal`/`geo`/`agri`/`econ`… |
| `type` | 과목 종류 | 아래 표 참고 |
| `semester` | 개설 학기 | `[1]`·`[2]`·`12`(매학기) |
| `recPick` | 고정 픽 순위 | 낮을수록 신청 성공률↑(선택) |
| `everyTerm` | 매학기 개설 | `true`(선택) |
| `division`/`divName` | 핵심교양 영역(교양 전용) | 6장 참고 |

**`type` 종류**

| type | 뜻 |
|---|---|
| `normal` | 일반 전공/선택 |
| `required_geo` | 주전공 전공필수 |
| `required_liberal` | 기초·중점교양 |
| `lang2` | 교양외국어(택1 그룹) |
| `div_liberal` | 핵심교양(영역별) |
| `basic_required` | 전공기초 |
| `S` | 패스/논패스(S/U) 과목 |

### 2-3. 카테고리 ↔ 탭 연결
과목의 `category` 값과, 수강신청 사이드바 탭의 `id`가 **같아야** 그 탭에 표시됩니다.

```js
// TAB_GROUPS (검색어: const TAB_GROUPS)
{id:'geo',  label:'사회교육과'},   // category:"geo" 과목이 여기 뜸
{id:'agri', label:'교직과정'},     // label만 바꾸면 탭 이름이 바뀜
```

## 3. 자주 쓰는 수정 레시피

**(A) 과목명만 바꾸기 — 가장 안전**
`name`만 고치면 연결이 그대로라 절대 깨지지 않습니다.
```js
{code:"208.205", name:"민주주의와 정치참여", ...},
```

**(B) 과목 추가**
해당 배열 안에 한 줄 추가. `code`만 겹치지 않으면 됩니다.
```js
{code:"SSE9001", name:"헌법과 인권", credit:3, category:"geo", type:"normal", semester:[1]},
```

**(C) 과목 삭제**
그 줄을 통째로 삭제. 단, 그 `code`가 `COURSE_SKILL_MAP`·`BOSS_DEFS`에 있으면 거기서도 함께 지우세요.

**(D) 과목을 다른 트랙으로 이동**
`category` 값을 기존 탭 `id` 중 하나로 변경.
```js
category:"geo"  →  category:"agri"   // 주전공 → 교직과정으로 이동
```

**(E) 신청 난이도/개설학기 조정**
`recPick`(낮을수록 쉬움), `semester`, `everyTerm`을 조정.

## 4. 교양(핵심교양) 시스템

인하대 교양 체계를 따릅니다.

| 탭 | type | 내용 |
|---|---|---|
| 기초·중점교양 | `required_liberal`·`lang2`·`basic_required` | 의사소통 영어, 커리어 디자인 1·2, 글쓰기, 크로스오버, 교양외국어 |
| 핵심교양 | `div_liberal` | 아래 6영역에서 4개 영역 택1 |
| 일반교양 | `div_liberal`(division 7) | SW·AI |

**핵심교양 영역**은 `division`(숫자) + `divName`(표시명) **두 값을 같이** 맞춰야 합니다.

| division | divName |
|---|---|
| 1 | 1영역 인간·가치·공존 |
| 2 | 2영역 역사·사상·문화 |
| 3 | 3영역 문학·예술·미디어 |
| 4 | 4영역 사회·제도·세계 |
| 5 | 5영역 자연·생명·환경 |
| 6 | 6영역 수리·정보·기술 |
| 7 | 7영역 일반교양 SW·AI |

```js
{code:"GED2001", name:"한국사의 이해", credit:3, category:"liberal",
 type:"div_liberal", division:2, divName:"2영역 역사·사상·문화", everyTerm:true},
```

## 5. 교수(보스) 수정

검색어: **`const BOSS_DEFS`**. 과목 `code`를 키로, 그 과목에 등장하는 교수 보스를 정의합니다.

```js
'M1707.000300': {
  name:'박선미', title:'교수', color:'#4ade80', size:28,
  quote:'설계 컨셉이 뭔가요?',          // 보스 대사
  ability:'rangedattack',               // 공격 패턴
  randomNames:['박선미','박선미','박선미'],
},
```
- 보스를 바꾸려면 `name`·`quote`·`randomNames`를 수정.
- 실제 교수진: **박선미(사회과교육)·정동준(정치)·장경호(경제)·김지훈(사회)·정상우(법)·김영순(문화)**.

## 6. 스킬 수정

- **스킬 라벨**(화면 표시): 검색어 `label:'방법론'` 등. 내부 키(`통계`,`GIS`…)는 그대로 두고 `label`만 바꾸세요.
- **과목→스킬 연결**: 검색어 `COURSE_SKILL_MAP`. `code`를 키로 부여 스킬과 단계를 지정합니다.
```js
'M0000.029300':{통계:1, GIS:1},   // 이 과목 이수 시 통계1·GIS1 해금
```

---

## 7. 탭/메뉴 구조

검색어 **`const TAB_GROUPS`**. 헤드(그룹)와 탭 목록을 정의합니다.
```js
{ head:'교직과정', tabs:[ {id:'agri', label:'교직과정'} ]},
{ head:'복수전공·부전공', tabs:[ {id:'econ', label:'경제학과'}, ... ]},
```
- 탭을 **이동**하려면 `{id:..., label:...}` 줄을 다른 `head` 그룹으로 옮기면 됩니다.
- **새 카테고리를 통째로 신설**하려면: ①탭 추가 ②그 `category`를 가진 과목 추가 ③`ALL_COURSES_WITH_POLISCI` 합치는 줄에 새 배열 포함. (고급 작업 — 기존 카테고리 재사용을 권장)

## 8. 대학원·진로 분기

- **대학원 학과**: 검색어 `GRAD_DEPTS`. `masterTeps`/`phdTeps`(어학 요건), `majorCat`(인정 전공) 조정.
- **진로 분기 카드**: 졸업 시 4갈래(대학원→교수 / 임용고시(사회) / 공기업 / 7급 공무원). 검색어 `lawschool-title`, `chooseCareer`.

## 9. 졸업요건 로직

검색어 **`function checkGraduationReady`** — 실제 졸업 가능 여부를 판정합니다.
```js
if(!passed('LIB002')) return false;        // 의사소통 영어
if(!passedType('lang2')) return false;     // 교양외국어
...
if(!geoIntensive && doubleMajors< 1) return false;  // 단일전공 고학점 또는 복수전공
```
화면의 졸업요건 체크리스트는 검색어 **`reqRow`** 로 그려집니다. 표시 항목 추가·문구 변경은 여기서 합니다.

## 10. 자주 깨지는 부분 (디버깅)

| 증상 | 원인 | 해결 |
|---|---|---|
| 게임이 흰 화면 | JS 문법 오류(쉼표·따옴표 빠짐) | F12 콘솔의 줄 번호 확인 후 수정 |
| 과목이 탭에 안 보임 | `category` ≠ 탭 `id` | 두 값을 일치시키기 |
| 신청해도 항상 실패 | `recPick`/`minPick` 설정 | 값을 낮추기 |
| 스킬·보스가 안 뜸 | `code`를 바꿔 연결이 끊김 | 이름만 바꾸거나 연결 키도 함께 수정 |
| 한글 깨짐 | 파일 인코딩 | **UTF-8**로 저장 |

## 11. 권장 작업 순서

1. **백업** 파일 복사
2. 바꿀 대상의 **검색 앵커**(const 이름)로 위치 찾기
3. **이름(`name`/`label`)만** 바꾸는 작업부터 (가장 안전)
4. 저장 → 브라우저 **새로고침** → 게임 시작해 확인
5. 문제 시 **F12 콘솔** 확인 → 안 되면 백업 복구

> 💡 표(엑셀)에서 과목을 편집하면 코드에 자동 반영해 주는 변환 도구가 필요하면 요청하세요. 직접 코드를 만지지 않고도 과목명·학점·카테고리를 안전하게 바꿀 수 있습니다.
